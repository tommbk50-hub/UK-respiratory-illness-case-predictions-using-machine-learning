England Influenza Forecast Dashboard

This dashboard provides a comprehensive 52-week forecast for Influenza in England across four critical metrics: Transmission (PCR Positivity), Primary Care Impact (GP Consultations), and Secondary Care Impact (Hospital & ICU Admissions).

Predictions are generated using a HistGradientBoostingRegressor hybrid model. The system is fully automated: it retrieves live data directly from the UKHSA public API and retrains all four models every week via GitHub Actions.

Technical Methodology

Our forecasting engine utilises a Hybrid Approach. For each metric independently, we first isolate the predictable seasonal cycle, and then use Scikit-Learn's HistGradientBoostingRegressor to model the residuals.

Source Reference: Scikit-Learn User Guide: Histogram-Based Gradient Boosting
